# Política de Seguridad

En Chronosfera no recopilamos datos personales identificables. Los sensores se leen de forma local y la única conexión externa es la API pública y anónima de la NOAA (Space Weather Prediction Center).

Si encuentras una vulnerabilidad de seguridad que pueda comprometer la integridad del código o la privacidad del usuario en un dispositivo físico, por favor contacta directamente al Director Científico del proyecto en lugar de abrir un Issue público.
